## 📚 Overview

A comprehensive, organized collection of **free TryHackMe rooms** with 500+ challenges categorized by learning path and difficulty level. This repository serves as a structured guide for cybersecurity enthusiasts to learn and progress through practical security challenges.

## 🎯 Purpose

- 📖 **Structured Learning**: Rooms organized by topic and difficulty
- 🎓 **Beginner-Friendly**: Start with basics and progress to advanced topics
- ✅ **Comprehensive**: 500+ free rooms all in one place
- 🚀 **Hands-On Practice**: Practical cybersecurity skills development

## 🚀 Getting Started

### Prerequisites
- TryHackMe Account (Free)
- Internet Connection
- VPN for Some Rooms

### Quick Start
1. Visit [TryHackMe](https://tryhackme.com)

3. Select a room from categories below
4. Follow the tasks
5. Submit flags
